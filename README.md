**Reloom** isn't just a contact book—it's a living, breathing personal CRM designed to help you weave the threads of your relationships with intention. With an elegant "Ivory & Satin" aesthetic, Reloom brings a premium, professional feel to managing your personal connections, memories, and important dates.

The best part? **It is 100% yours.** There is no cloud sync, no tracking, and no hidden backend. Everything lives securely inside a SQLite vault directly on your device.

---

## ✨ Features

- 🛡️ **Absolute Privacy (Local-First):** Your data never leaves your phone. Powered by a high-performance local SQLite database and Drizzle ORM, ensuring maximum security and offline capability.
- 👥 **Dynamic Social Web:** Move beyond simple contacts. Map out how people are connected to each other, tag friends in shared journal memories, and organize your network spatially by cities.
- 📓 **Semantic Journaling:** Log interactions, gift ideas, and shared memories with an integrated markdown editor. Keep the context of your relationships rich and accessible.
- 📅 **Temporal Intelligence:** A built-in calendar seamlessly aggregates birthdays, anniversaries, and custom reminders so you never miss an important milestone.
- 🎨 **Premium Aesthetic:** Designed with a stunning, tactile UI that feels responsive and alive. Smooth gradients, glassmorphism, and curated typography make organizing your life a joy.

## 🚀 Built With

Reloom leverages a modern, robust technology stack:
- **Framework:** React Native / Expo
- **Database:** Expo SQLite + Drizzle ORM
- **Navigation:** Expo Router
- **Animations:** Reanimated & Skia (for physics-based, butter-smooth interactions)

## 💻 Local Development

Want to compile it yourself or contribute? Getting started is easy.

```bash
# 1. Clone the repository
git clone https://github.com/voilry/reloom.git

# 2. Navigate to the app directory
cd reloom/Reloom

# 3. Install dependencies
npm install

# 4. Start the development server
npx expo start
```
