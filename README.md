# Reloom

Reloom is a privacy-first personal CRM and social registry. It is designed to help you keep track of your relationships, interactions, and important dates—all stored 100% locally on your device.

## Why Reloom?
- **True Privacy:** Your data never leaves your device. There are no servers, no cloud sync, and no tracking.
- **Rich Context:** Add detailed notes, track how you met someone, and categorize your connections.
- **Integrated Journal:** Write memories and tag people you spent time with.
- **Smart Reminders:** Get notified of birthdays, anniversaries, and custom reminders natively.

## Tech Stack
- Built with **React Native** & **Expo**.
- Local database powered by **SQLite** and **Drizzle ORM**.
- Custom "Ivory & Satin" design system for a premium UI.

## Local Setup
To run the app locally:

```bash
cd Reloom
npm install
npx expo start
```
